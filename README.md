# Machine Learning Pipeline with Scikit-learn

An end-to-end machine learning pipeline demonstrating data preprocessing, model training, evaluation, and prediction using the Iris dataset.

## Project Overview

This project covers the standard workflow followed in machine learning projects.

The pipeline includes:

- Loading the dataset
- Exploratory Data Analysis (EDA)
- Train-test split
- Feature scaling
- Logistic Regression model training
- Performance evaluation
- Predictions on new samples
- Model persistence using Joblib

## Technologies Used

- Python
- NumPy
- Pandas
- Scikit-learn
- Matplotlib
